## 📱 О проекте

**ПроСТО** — это веб-приложение для управления автосервисом, которое делает ремонт прозрачным и удобным для всех участников: клиентов, механиков и владельца.

### Ключевая идея
> «Система с красивым клиентским приложением и понятным веб-кабинетом для мастера, где главным инструментом контроля станут фотоотчеты и чек-листы, а не сложный складской учет. Владельцы СТО получат прозрачный бизнес, а клиенты — доверие и удобство, за которые они готовы возвращаться и рекомендовать сервис.»

### Основные возможности

#### Для клиента
- 📅 Онлайн-запись на сервис с выбором услуги, даты и времени
- 📸 Просмотр фотоотчета после выполнения работ
- 📋 Отслеживание статуса заказа (ожидание → подтверждение → работа → завершён)
- 💳 История всех ремонтов с детализацией по услугам и запчастям
- 🚗 Управление списком автомобилей

#### Для механика
- 📋 Дашборд с задачами на день
- ✅ Чек-лист выполнения работ с пошаговой отметкой
- 📸 Загрузка фотоотчета (старая/новая деталь, процесс)
- 🔧 Добавление использованных запчастей и материалов
- 🎯 Возможность брать доступные заказы в работу

#### Для владельца
- ⚙️ Управление услугами (создание, редактирование, цены, чек-листы)
- ✅ Подтверждение записей клиентов
- 👨‍🔧 Назначение механиков на заказы (опционально)
- 📊 Контроль выполнения работ через фотоотчеты

## 🚀 Быстрый старт

### Локальный запуск

```bash
# Клонирование репозитория
git clone https://github.com/MPaidaKI23/ISIS.git
cd ISIS/prosto

# Создание виртуального окружения
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Установка зависимостей
pip install -r requirements.txt

# Применение миграций
python manage.py migrate

# Создание суперпользователя
python manage.py createsuperuser

# Запуск сервера
python manage.py runserver
